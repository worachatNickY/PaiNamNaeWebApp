## [1.0.3] — 2025-03-12
Added
Chat System Enhancement
เพิ่มระบบส่งข้อความแบบ Real-time ภายในแอป
รองรับการส่ง:
- รูปภาพ (Media)
- ตำแหน่ง (Location)
เพิ่ม Privacy Confirmation Modal ก่อนแชร์ข้อมูลสำคัญ

Trip Interaction
เพิ่มระบบแสดงสถานะการเดินทาง (Trip Status Tabs)
รองรับการใช้งานแชทระหว่างการเดินทาง (In-ride Chat Actions)

Fixed
แก้ไขปัญหาที่เกิดจากการรวมโค้ด (Merge Conflict)
ปรับปรุงการแสดงผล UI ของ Chat และ Modal

Changed
ปรับปรุง UI/UX ของระบบ Chat ให้ใช้งานง่ายขึ้น
ปรับปรุงการจัดการสถานะการเดินทาง
ปรับโครงสร้างบางส่วนเพื่อรองรับฟีเจอร์ใหม่

Technical Changes
Backend:
- ปรับปรุง logic รองรับ media และ location ใน chat
- อัปเดตโครงสร้างข้อมูลของระบบ Chat

Frontend:
- ปรับปรุง UI Chat และ Interaction
- เพิ่ม modal สำหรับยืนยันการแชร์ข้อมูล
