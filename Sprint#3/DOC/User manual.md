User Manual – ระบบ Chat ระหว่าง Driver และ Passenger
1. ภาพรวมระบบ

    ระบบ Chat นี้ถูกออกแบบเพื่อให้ Driver และ Passenger สามารถสื่อสารกันได้อย่างปลอดภัย ระหว่างการเดินทาง โดย:

        ไม่เปิดเผยข้อมูลส่วนตัว

        รองรับการส่งข้อความ / รูปภาพ / ตำแหน่ง

        มีระบบป้องกันข้อผิดพลาดและความปลอดภัย

2.  ผู้ใช้งานระบบ

        Driver (คนขับ)

        Passenger (ผู้โดยสาร)

3.  การใช้งานฟีเจอร์ Chat
    3.1 ส่งข้อความ

        เปิดหน้า Chat ระหว่าง Ride

        พิมพ์ข้อความ

        กดปุ่ม Send

        ข้อความจะถูกส่งและแสดงทั้ง 2 ฝั่ง

    3.2 ตอบกลับข้อความ

        Passenger สามารถพิมพ์และกด Send เพื่อตอบกลับได้ทันที

    3.3 ข้อจำกัดของข้อความ

        ห้ามส่งข้อความว่าง
        → ระบบจะแสดง: "Message cannot be empty"

        ห้ามส่งหลัง Ride เสร็จ
        → ระบบจะแสดง:
        "Chat is no longer available after the ride has been completed."

4. ความปลอดภัยของข้อมูล

    ระบบจะ ไม่แสดงข้อมูลส่วนตัว เช่น:

        เบอร์โทรศัพท์

        Email

        ที่อยู่

    และยังมีระบบ:

        ตรวจจับข้อมูลส่วนตัวในข้อความ

        บล็อกข้อความที่ไม่ปลอดภัย

5.  การส่ง Location
    วิธีส่ง:

        กด Send Location

        ระบบจะแสดงข้อความยืนยัน:

        “Do you want to share your location with the passenger?”

        กด Confirm → ส่งสำเร็จ

    ยกเลิก:

        กด Cancel → ไม่ส่งตำแหน่ง

6.  การส่งรูปภาพ
     วิธีส่ง:

        เลือกรูปภาพ

        ระบบจะแสดง:

        “Are you sure you want to send this image?”

        กด Confirm → ส่งสำเร็จ

    ยกเลิก:

        กด Cancel → ไม่ส่งรูป

7. 💾 การบันทึกข้อความ

        ข้อความทั้งหมดจะถูกบันทึกในระบบ

        ผู้ใช้สามารถกลับมาเปิดดูย้อนหลังได้

8. ⚙️ เงื่อนไขการใช้งานระบบ

        ใช้งานได้เฉพาะตอน Ride = active

        ถ้า Ride = completed
        → จะไม่สามารถแชทได้
